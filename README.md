# Google 商家 / LINE OA 店主 SOP 套件

這個資料夾是給店主與代辦方共同使用的交付包。

用途分成兩個層次：

1. 給店主：只看 `01-給店主的完整SOP.md`、`02-店主需提供的欄位與文案.md`、`03-店主需提供的素材清單.md`
2. 給你：看 `04-你代辦處理的範圍與重點.md`，快速知道哪些事情你可以直接做、哪些一定要店主本人定稿

  - https://lineforbusiness.com/th-en/helpcenter/line-oa

## 檔案說明

- `01-給店主的完整SOP.md`
  - 店主要做什麼、先後順序、授權方式、驗證時要配合什麼
- `02-店主需提供的欄位與文案.md`
  - Google 商家與 LINE OA 需要的欄位資料與必要文案
- `03-店主需提供的素材清單.md`
  - 照片、影片、Logo、頭像、封面、驗證素材的規格與拍攝要求
- `04-你代辦處理的範圍與重點.md`
  - 你可代做的工作、需要店主決策的內容、最重要的風險提醒
- `05-可直接傳給店主的訊息模板.md`
  - 可直接複製貼上給店主的訊息

## 建議使用方式

1. 先把 `05-可直接傳給店主的訊息模板.md` 傳給店主
2. 請店主依 `02` 與 `03` 回傳資料
3. 你收到資料後，依 `04` 進行 Google 商家與 LINE OA 建置
4. 最後只保留需要店主本人確認或授權的步驟給店主操作

## 這份套件的核心原則

- 店主不需要理解所有平台細節，只需要完成授權、提供真實資料、確認最終對外文案
- 所有涉及真實世界身分、商家名稱、地址、電話、營業資訊、招生承諾、價格與資格聲明的內容，都以店主最終確認為準
- 所有需要平台管理權限的操作，優先使用「加管理者 / 管理員」方式，不共用密碼
