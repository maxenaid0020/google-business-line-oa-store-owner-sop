# 店主需提供的素材清單

這份清單只列出真正值得向店主要的素材。

原則：

- 先求真實可用，再求好看
- Google 驗證素材優先於設計素材
- 能一次拍高解析原圖，就不要只傳截圖或壓縮圖

---

## A. Google 商家必備素材

### 1. 驗證素材（最高優先）

請店主至少提供：

- 店外觀 3 張
- 門牌 / 招牌 / 大樓入口 3 張
- 教室 / 辦公 / 接待空間 3 張
- 教材、工具、設備 3 張
- 可證明商家真實存在的名片、文件、識別物 2 張

### 2. 上架照片素材（高優先）

請店主至少提供：

- 品牌主圖 1 張
- 場地外觀 3 張
- 場地內部 3 張
- 教學情境 6 張
- 實作 / 施工情境 6 張
- 團隊 / 講師照片 3 張

### 3. Google 商家照片建議規格

- 檔案格式：`JPG` 或 `PNG`
- 上傳尺寸：建議至少 `720 x 720`
- 最低可接受：`250 x 250`
- 檔案大小：`10 KB` 到 `5 MB`

建議店主交給你時：

- 原始檔優先
- 單張長邊至少 `2000 px`
- 不要用社群截圖
- 不要用過度濾鏡圖

### 4. Google 商家影片建議

- 長度：`30 秒` 內
- 解析度：至少 `720p`
- 檔案大小：`75 MB` 內

建議優先收：

- 實作教學片段
- 教室 / 上課環境導覽
- 講師教學片段
- 真實施工過程片段

---

## B. LINE OA 必備素材

### 1. 頭像

- 建議提供：Logo 檔或清楚品牌圖樣
- 建議輸出尺寸：`640 x 640`
- 格式：`JPG` / `JPEG` / `PNG`
- 檔案大小：`3 MB` 內

注意：

- 頭像通常以圓形顯示
- 不要塞太多字
- 最好一眼能辨識品牌

### 2. Profile Page 封面

- 建議輸出尺寸：`1080 x 720`
- 格式：`JPG` / `JPEG` / `PNG`
- 檔案大小：`3 MB` 內

適合放的內容：

- 品牌主視覺
- 教學現場照
- 一句核心主標

### 3. Rich Menu

- 大版建議：`2500 x 1686`
- 小版建議：`2500 x 843`
- 也可接受：`1200 x 810`、`1200 x 405`、`800 x 540`、`800 x 270`
- 格式：`JPG` / `JPEG` / `PNG`
- 檔案大小：`1 MB` 內

建議先決定 3 到 6 個主按鈕：

- 課程介紹
- 課程費用
- 上課地點
- 講師介紹
- 報名方式
- 一對一諮詢

### 4. Rich Message

- 方形建議：`1040 x 1040`
- 長形自訂：寬 `1040`，高 `520` 到 `2080`
- 格式：`JPG` / `JPEG` / `PNG`
- 檔案大小：`10 MB` 內

建議至少做：

- 1 張課程總介紹
- 1 張講師信任圖卡
- 1 張招生 CTA 圖卡

---

## C. 店主交素材時的實際要求

請店主不要只說「你自己挑」。

請店主交付時至少做到：

1. 每張照片知道拍的是什麼
2. 確認是否可對外公開
3. 盡量提供原檔
4. 不要混入不想曝光的人臉或私人資訊
5. 如果照片裡有學員，先確認是否可公開使用

---

## D. 建議店主的素材資料夾結構

店主可以直接把資料放成這樣：

```text
店家素材包/
  01-店外與招牌/
  02-教室與空間/
  03-教學現場/
  04-施工與實作/
  05-講師照片/
  06-Logo與品牌/
  07-影片/
  08-驗證補充資料/
```

---

## E. 最值得店主親自拍的 10 張照片

如果店主時間很少，至少先補這 10 張：

1. 店門口全景
2. 招牌近照
3. 大樓入口或街景辨識照
4. 教室全景
5. 實作區全景
6. 講師教學中照片
7. 學員實作中照片
8. 工具 / 設備整齊陳列照片
9. 品牌名片或文件照片
10. 一張最能代表課程價值的主視覺照片
