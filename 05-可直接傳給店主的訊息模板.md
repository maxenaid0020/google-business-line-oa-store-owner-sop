# 可直接傳給店主的訊息模板

你可以直接複製下面這段傳給店主。

---

嗨，我這邊會幫你整理 Google 商家 / Google 地圖資訊，以及 LINE 官方帳號的內容與素材。

你這邊不用研究平台細節，主要只需要配合 3 件事：

1. 授權我管理 Google 商家與 LINE OA
2. 提供真實資料、照片、影片與課程資訊
3. 最後確認對外公開的內容

為了方便操作，先附上官方入口：

- Google 商家申請 / 新增商家： https://business.google.com/add
- Google Business Profile： https://business.google.com/
- Google 商家管理權限說明： https://support.google.com/business/answer/3403100?hl=en
- LINE 官方帳號申請： https://entry.line.biz/start/jp/
  - 備註：此入口為官方頁面，但常會顯示日文介面
- LINE Official Account Manager： https://manager.line.biz/
- LINE OA 說明中心： https://lineforbusiness.com/th-en/helpcenter/line-oa

為了避免之後反覆來回，麻煩你一次準備以下資料給我：

## 1. 請先告訴我這些基本資料

- 正式店名
- Google 地圖想顯示的名稱
- 是否有固定地址
- 完整地址
- 對外電話
- 可聯絡時間 / 可接待時間
- 官方網站 / 課程頁 / 報名表單
- 是否已有 LINE 官方帳號
- 若已有，請給我加好友連結

## 2. 課程資料請提供

- 課程正式名稱
- 課程費用
- 總時數
- 線上 / 線下各幾堂
- 上課地點
- 適合對象
- 課程主要教什麼
- 是否有實作
- 結業後是否提供媒合 / 介紹工作 / 創業協助
- 報名流程

## 3. 講師資料請提供

每位講師至少要有：

- 姓名
- 對外頭銜
- 現職
- 從業年資
- 可公開的證照 / 經歷
- 可公開的講師照片

## 4. 請提供以下照片 / 影片原檔

- 店外觀
- 招牌 / 門牌 / 大樓入口
- 教室或上課空間
- 教學現場
- 實作 / 施工現場
- 講師照片
- Logo 或品牌圖樣
- 可公開使用的影片

## 5. 權限授權

### Google 商家

如果你還沒有 Google 商家，先到 `https://business.google.com/add` 申請。

請用新增管理者 / 擁有者的方式授權我，不需要把帳密給我。

操作路徑：

1. 登入 `https://business.google.com/`
2. 進入商家檔案
3. 打開 `Business Profile settings`
4. 進入 `People and access`
5. 把我加成 `Owner` 或 `Manager`

### LINE OA

如果你還沒有 LINE 官方帳號，先到 `https://entry.line.biz/start/jp/` 申請。

請在 LINE Official Account Manager 裡把我加成管理員，不需要直接給我個人 LINE 帳密。

操作入口：

- `https://manager.line.biz/`

## 6. 最後你只需要確認這些內容

- 店名
- 地址
- 電話
- 課程費用
- 課程時數
- 講師介紹
- 哪些照片 / 影片可公開
- 對外承諾是否正確

你如果願意，我可以再把這份清單整理成你只要逐項填空的版本，讓你更好回傳。

---

如果店主很忙，也可以再傳這句補充：

`你不用自己寫得很漂亮，只要把真實資訊、照片與原始答案給我，我會幫你整理成對外版本。`
